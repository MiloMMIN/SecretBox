# 智心树洞 v2.1 更新说明 (2026-03-13 ~ 2026-03-24)

### 🌟 核心业务功能
- 新增：教师邀请码机制，配置 `TEACHER_INVITE_CODE` 后，教师可在小程序内自助输入邀请码升级为教师身份，无需后台手动维护。
- 新增：教师个人主页管理页（counselor_manage），支持头像上传、展示名称 / 简介编辑、在线状态开关，教师可独立维护自己的公开资料。
- 新增：广场内容管理后台（square_manage），教师可查看、审核及删除广场问题，管理全站公开内容。
- 优化：用户角色识别与登录流程，完善多角色权限分发逻辑，修复登录后角色信息未及时刷新的问题。

### 🎨 体验与前端交互
- 优化：树洞对话页交互体验，消息流布局更清晰，发送状态反馈更即时。
- 优化：内容审核流程前端展示，将审核状态提示精简至必要节点，减少干扰信息。
- 重构：首页提交入口、问题创建页对话加载逻辑及个人中心教师问题分页，统一移除冗余 UI 状态，页面更轻量流畅。
- 移除：问题详情页中面向用户的审核状态字段展示（审核逻辑后移至服务端静默处理）。

### 🛠️ 后端架构与系统稳定性
- 优化：`init_db.py` 增加缺失字段自动检测与补全逻辑，数据库升级更安全，兼容存量数据。
- 优化：服务端问题审核逻辑精简，去除不必要的 `audit_status` / `review_reason` 字段透传，降低 API 复杂度。
- 新增：`tools/load_test.py` 压测工具，可对核心接口进行并发基准测试，便于上线前性能验证。
- 重构：精简 `app.py` 中与已下线功能相关的路由及 schema 定义，代码体积减少约 30%，可维护性提升。

# 智心树洞 v2.0 发布说明 (Release Notes)

基于当前项目的最新迭代状态与已落地的所有核心闭环能力，以下是 v2.0 版本的正式更新内容记录：

### 🌟 核心业务功能
- 新增：问题广场功能，支持公开提问与互动回答，打造校园心理互助社区。
- 新增：匿名树洞（私密留言）功能，支持向指定教师或全体星空树洞进行匿名/实名倾诉，全方位保护学生隐私。
- 新增：教师工作台与管理后台，支持查看专属留言、数字红点待办提醒，以及一键导出近期对话记录留存。
- 新增：多级角色与灵活权限体系，提供专属“教师邀请码”，支持符合资质的教师在小程序内输入后自助升级权限。
- 新增：自动化内容安全保障，全量接入微信官方内容安全审核 API（涵盖文本、图片发布），实时拦截敏感风险内容。

### 🎨 体验与前端交互
- 新增：视觉防错交互设计，明确问题广场（中性灰色系）与私密树洞（安全绿色系）的 UI 视觉隔离，降低学生误发公开区域的风险。
- 新增：全站（首页广场、选择倾诉对象、个人中心详情页等）全局头像加载容错机制。当外部图片失效时，智能平滑降级为“默认昵称首字母提取”的文字占位头像。

### 🛠️ 后端架构与系统稳定性
- 新增：后端图片路径自动重组解析能力，智能兼容历史遗留的旧域名数据或错误格式的相对路径。
- bugfix：修复由于生产环境（Nginx 反向代理）配置与后端路由拼接不匹配，导致用户新上传头像及回复图片后出现 404 无法访问的问题（统一路由配置适配 `/api/uploads/` 代理策略）。
- bugfix：修复因图片失效或网络波动，导致的小程序多处列表页及详情页出现的排版错位、空白卡顿问题。
- bugfix：修复 `build_file_url` 在多环境下（本地测试与线上云服务器）绝对路径生成不一致、遗留无效域名的历史脏数据访问异常等问题。
