
## 项目概述

智心树洞是一个微信小程序，为温州科技职业学院数智技术学院提供学生与教师之间的沟通渠道。核心价值：让学生敢于说真话，帮助教师及时发现心理问题学生。

**目标用户：**
- 学生：大专生，来自五湖四海，经济背景多样
- 教师：工作繁忙，需要快速识别极端心理问题

**平台：** 学院公众号中安插一个微信小程序（取代问卷星）

## 核心功能

### V1.0 MVP（当前版本）

**1. 问题广场（公开 Q&A）**
- 展示公开问题和回答
- 师生都可以提问和回答
- 实时内容审核（微信内容安全 API）
- UI：灰色系，公共区域视觉标识

**2. 匿名树洞（私密留言）**
- 学生选择教师进入树洞
- 支持匿名/实名留言（仅文字）
- 选择是否公开到广场
- 教师私密回复（不公开）
- UI：绿色系，私密区域视觉标识

**功能描述：**
页面首页是一个框框--问题广场，可以把公开的问题在这里展示，框框下面有一个信箱的图标，点进去就可以开始投稿，点击之后显示学工办所有教师老师，每一个老师后面都有进入树洞的按钮，问题描述可以通过文字或者语音录制的方式，下方可以选择是否公开在广场，是否匿名提问。

**3. 教师端功能**
- 查看分配给自己的树洞留言
- 查看学生真实身份（即使匿名）
- 钉钉 Webhook 通知
- 一键导出对话记录（30天数据保留）
- 在问题广场回答问题

**关于留言接收：** 可采用微信推送的方式（服务号？），或者给所有老师一个权限可以进后台看看留言。

**4. 权限管理**
- 对应教师可查看自己的留言
- 高权限管理员可查看所有留言
- 学生可删除自己的留言

### V2.0 规划（未来版本）

- AI 预筛选和心理评估
- 语音留言功能
- 心理健康资源库
- 数据分析看板（热点话题）
- 预约教师面谈功能

## 技术架构

### ✅ 技术栈

**前端：**
- 微信小程序（WXML + WXSS + JavaScript）

**后端：**
- Python Flask + SQLAlchemy（ORM）
- Gunicorn（WSGI 服务器）
- Celery + Redis（异步任务队列，用于钉钉通知等）

**数据库：**
- MySQL 8.0（主数据库）
- Redis（缓存 + 消息队列）

**集成服务：**
- 微信内容安全 API（实时审核）
- 钉钉 Webhook（教师通知）
- 微信公众号（小程序入口）

**部署：**
- Docker + Docker Compose
- 云服务器（阿里云/腾讯云）

**选择理由：**
- Python 生态适合未来 AI 功能扩展（V2.0 心理评估）
- Flask 轻量灵活，适合快速 MVP 开发
- MySQL 结构化存储，适合用户/留言管理
- 技术栈成熟稳定，便于学生接手维护

- 符合大众审美的配色方案
- 温暖、安全、可信赖的视觉风格

## 风险与阻塞

### 伦理与法律风险
- **假匿名问题**：学生以为匿名，但教师能看到真实身份
  - 风险：学生信任度下降，可能引发投诉
  - 缓解：仅在紧急情况（自伤风险）时查看身份，建立明确的使用规范

- **数据隐私合规**：学生心理健康数据属于敏感信息
  - 风险：违反数据保护法规
  - 缓解：咨询法务部门，确保符合《个人信息保护法》

### 技术风险
- **内容审核误判**：AI 审核可能误判正常内容或漏掉敏感内容
  - 缓解：建立人工复审机制，设置敏感词库

- **极端情况响应**：学生表达自伤倾向时的紧急处理
  - 缓解：建立紧急通知机制（钉钉 + 短信），制定应急预案

### 运营风险
- **使用率低**：学生不信任或不使用平台
  - 缓解：初期宣传推广，保护学生隐私，建立信任

- **问题广场冷清**：没有人提问或回答
  - 缓解：教师主动发起话题，设置激励机制（积分/徽章）

**前端：微信小程序**

**后端选型考虑：**
- Node.js：快速开发，适合实时通信
- Python Flask：轻量级，适合 MVP
- Java Spring：企业级，适合长期维护

**数据库选型：**
- MySQL：结构化数据，适合用户、留言等
- MongoDB：灵活存储，适合对话记录

**集成服务：**
- 微信内容安全 API：实时审核
- 钉钉 Webhook：教师通知
- 微信公众号：小程序入口
